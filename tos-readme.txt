Game ini di design dengan pygame, cara memainkannya harus di download dulu source codenya,
buka main.py

navigasi game menggunakan arrow up, down, left, right ,dan space untuk lompat.

tujuan dari game ini adalah mendapatkan score setinggi mungkin, kami menyimpan high score dengan nama user yang memainkannya.


How to play
1. start app
2. masukan nama
3. token player_name ditampilkan sebagai nama, high score adalah dari data yang pernah ada
4. start game
5. jalan pakai panah, lompat pakai spasi (hold untuk lompat yang  lebih tinggi)
6. powerup boost berupa lompat sangat tinggi bila diambil
7. bila mencapai hiigh score maka akan me replace high score sebelumnya
8. bila tidak high score ditamiplkan sebagai score saja.
